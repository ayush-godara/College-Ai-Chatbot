# 🎓 College AI Chatbot

An AI-powered chatbot that answers questions about **BMS College of Engineering (BMSCE), Bangalore**.

This project provides a simple web interface where students can ask questions related to the college, and the chatbot generates intelligent responses using a Large Language Model.

---

## 🚀 Live Demo

Try the chatbot here:

👉 https://bmsce-ai-chatbot.streamlit.app

---

## 🧠 Features

- AI chatbot for college-related queries
- Web-based interface built with Streamlit
- Uses LLM to generate answers
- Simple and interactive UI
- Accessible from any browser

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **Groq API (Llama models)**
- **DuckDuckGo Search**
- **BeautifulSoup (web scraping)**
- **Requests**
- **dotenv**

## ⚙️ Installation (Local Setup)

1. Clone the repository
git clone https://github.com/ayush-godara/College-Ai-Chatbot.git
2. Install dependencies
pip install -r requirements.txt
3. Create `.env` file
   GROQ_API_KEY=your_api_key
4. Run the app
   streamlit run app.py
